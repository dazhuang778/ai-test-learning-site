resources:
  - title: GitHub Copilot
    url: https://github.com/features/copilot
    type: tool
    description: 微软/GitHub 出品，深度集成主流 IDE，代码补全和测试生成能力强，适合日常编码与测试开发。
  - title: Cursor IDE
    url: https://cursor.com/
    type: tool
    description: AI-first 代码编辑器，支持对话式代码生成，对测试文件的完整生成和重构效果突出。
  - title: Tabnine
    url: https://www.tabnine.com/
    type: tool
    description: 企业级 AI 代码助手，支持私有化部署，适合对数据安全有要求的团队使用。
  - title: AI 测试工具全景探索 - Ministry of Testing
    url: https://www.ministryoftesting.com/articles/an-exploration-of-10-ai-software-testing-tools
    type: article
    description: Ministry of Testing 对 10 款主流 AI 测试工具的深度探索，涵盖用例生成、自动化、视觉测试等各类工具评测。
  - title: Windsurf（前 Codeium）
    url: https://windsurf.ai/
    type: tool
    description: 新一代 AI 代码编辑器，内置 Cascade AI 深度上下文理解，测试代码生成和跨文件重构能力突出。
  - title: Continue.dev - 开源 AI 代码助手
    url: https://www.continue.dev/
    type: tool
    description: 完全开源的 VS Code / JetBrains AI 编程助手，支持接入任意 LLM，适合希望自主控制 AI 工具链的团队。
  - title: Codeium - 免费 AI 代码助手
    url: https://codeium.com/
    type: tool
    description: 免费 AI 代码补全工具，支持 70+ 编程语言和 40+ IDE，个人用户永久免费，适合学习阶段使用。
  - title: Qodo AI（原 CodiumAI）
    url: https://www.qodo.ai/
    type: tool
    description: 专注代码质量与测试的 AI 工具，根据代码逻辑自动生成有意义的单元测试，是测试工程师的专项利器。
  - title: Applitools - 视觉 AI 测试
    url: https://applitools.com/
    type: tool
    description: 业界领先的视觉 AI 测试平台，利用深度学习对比 UI 截图差异，大幅减少视觉回归测试的人工成本。
  - title: Sauce Labs - 云端测试平台
    url: https://saucelabs.com/
    type: tool
    description: 跨浏览器和移动端云测试平台，内置 AI 分析失败原因，支持 Selenium/Playwright/Appium 等主流框架。
  - title: BrowserStack - 真机云测试
    url: https://www.browserstack.com/
    type: tool
    description: 提供真实设备云测试环境，内置 AI 测试可观测性（Test Observability），智能分析测试失败和不稳定测试。
  - title: Devin - AI 软件工程师
    url: https://www.cognition.ai/blog/introducing-devin
    type: tool
    description: Cognition 推出的自主 AI 软件工程师，可独立完成编写测试、修复 Bug 等复杂任务，代表 AI 测试智能化的前沿方向。

---

工具选型建议：个人学习推荐 GitHub Copilot（生态最成熟）；追求对话式生成推荐 Cursor；企业私有化部署推荐 Tabnine 或 Continue.dev；视觉测试推荐 Applitools；云端跨浏览器测试推荐 BrowserStack。
