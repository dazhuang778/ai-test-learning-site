## ADDED Requirements

### Requirement: 知识图谱节点卡片渐变背景
知识图谱中的每个节点卡片 SHALL 展示渐变背景，以视觉方式区分不同层级的节点。

#### Scenario: 节点卡片展示层级对应渐变背景
- **WHEN** 知识图谱渲染完成
- **THEN** 每个节点卡片展示与其 Level 对应的渐变背景色（如 Level 0 为蓝绿色系，Level 1 为蓝色系，Level 2 为紫色系，Level 3 为玫红色系）

#### Scenario: 渐变背景在图谱缩放后保持正确
- **WHEN** 用户对图谱执行缩放操作
- **THEN** 节点卡片的渐变背景在各缩放比例下均正确渲染，无颜色失真或背景丢失

### Requirement: 知识图谱节点悬停上浮动画
知识图谱中的节点卡片 SHALL 在鼠标悬停时展示上浮动画，增强交互反馈。

#### Scenario: 鼠标悬停触发节点上浮
- **WHEN** 用户鼠标悬停在图谱节点卡片上
- **THEN** 节点卡片产生向上位移的动画效果（如 `translateY(-4px)` 或等效 CSS 过渡），并伴有阴影增强

#### Scenario: 鼠标离开后节点复位
- **WHEN** 用户鼠标从图谱节点卡片上移开
- **THEN** 节点卡片平滑过渡回原始位置，不影响相邻节点或连线的布局
