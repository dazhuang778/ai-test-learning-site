## Why

软件测试工程师在 AI 浪潮下面临技能升级压力，但现有学习资源分散、缺乏体系化引导。本项目旨在构建一个「AI辅助软件测试」知识图谱学习网站，帮助测试人员在30秒内看清学习全貌，并快速找到适合自己阶段的高质量外链资源。

## What Changes

- **新建** 可视化知识图谱主页，展示4层级、7节点的 AI 测试学习体系
- **新建** 知识节点详情页，展示每个主题的精选外链资源（文章/视频/课程/工具）
- **新建** Markdown 内容数据层，每个知识节点对应一个 `.md` 文件，存储于 Git 仓库
- **新建** 静态站点构建流水线，可部署至 Vercel 或 GitHub Pages

## Capabilities

### New Capabilities

- `knowledge-graph`: 可视化知识图谱，展示节点层级关系，支持点击导航至详情页
- `node-detail`: 知识节点详情页，展示节点简介、难度、阶段标签及精选资源列表
- `content-management`: Markdown 文件驱动的内容维护机制，支持增删改知识节点和资源链接

## Impact

- **内容结构**：`content/nodes/` 目录下存放所有知识节点 Markdown 文件
- **前端**：全新静态站点项目（技术栈由设计阶段确定），无后端依赖
- **部署**：静态文件，无需服务器，支持 Vercel / GitHub Pages 一键部署
- **依赖**：知识图谱可视化需引入图形渲染库（如 D3.js 或 React Flow）
